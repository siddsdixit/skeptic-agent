# Changelog

All notable changes to skeptic-agent are documented here. Format: [Keep a Changelog](https://keepachangelog.com/en/1.1.0/). Versioning: [SemVer](https://semver.org).

## [0.2.0] — 2026-04-27

Multi-runtime support, new README, real LICENSE.

### Added

- **`AGENTS.md`** — runtime adapter for Codex CLI, Cursor, Aider, OpenCode, and any tool that reads [agents.md](https://agents.md). Same six-pass methodology, runtime-portable.
- **`GEMINI.md`** — runtime adapter for Google Gemini CLI.
- **`.cursorrules`** — runtime adapter for legacy Cursor versions.
- **`LICENSE`** — MIT. Required for OSS adoption.
- **README rewrite** — viral-grade pitch with concrete use cases (PRDs, pitch decks, migrations, hiring, emails, trades, decisions, actions), example output snippet, multi-runtime install grid, philosophy, heritage section, principles.

### Changed

- **Positioning** — "the adversarial review agent" front and center. Use cases beyond code (strategy, email, pitch, migration, trade, decision) now leading.
